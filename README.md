# Personal Finance Tracker

A responsive personal finance management application built using **HTML, CSS, and JavaScript**. The project helps users track income and expenses, monitor spending habits, set monthly limits, and view financial statistics through an interactive dashboard.

---

## Features

### Dashboard

* View current balance
* Track total income and expenses
* Display recent transactions
* Quick access to common actions

### Transaction Management

* Add income and expense transactions
* Edit existing transactions
* Delete transactions
* Search transactions by category or description
* Filter transactions by income or expense

### Statistics

* Total number of transactions
* Average expense calculation
* Largest expense tracking
* Savings calculation
* Expense breakdown by category
* Income vs Expense summary

### Budget Control

* Set a monthly spending limit
* Visual warning when the limit is exceeded
* Monthly limit stored using local storage

### Settings

* Configure monthly spending limit
* Clear all stored transaction data
* Floating settings panel

### Theme Support

* Light and Dark mode
* Theme preference saved using local storage

### Data Persistence

* Transactions stored using browser local storage
* Data remains available after page refreshes

### New Month Feature

* Clears previous transactions
* Carries forward the current balance as an opening balance
* Helps manage finances month by month
